## ¿Qué es TaskMaster?

TaskMaster es una aplicación de escritorio para **Windows** diseñada para que organices tu trabajo y proyectos personales de forma clara y sin distracciones. No es una herramienta online, no requiere cuenta, y todo se guarda directamente en tu dispositivo.

Está pensada para quienes necesitan algo más potente que una lista de notas, pero sin la complejidad de herramientas empresariales. Rápida, offline-first, y con un diseño oscuro que no cansa la vista.

## Características principales

### 📋 Gestión de tareas completa
- Crea tareas con título, descripción, fecha límite y prioridad (**Urgente**, **Alta**, **Media**, **Baja**)
- Añade **subtareas** para dividir el trabajo en pasos más pequeños
- Marca tareas como completadas con un solo clic o atajo de teclado
- Menú rápido con **clic derecho** para cambiar estado, prioridad, copiar o eliminar sin abrir la tarea

### 🗂 Categorías y proyectos
- Organiza tus tareas en **categorías personales** y **proyectos**
- Cada proyecto tiene sus propias vistas: **Kanban**, **lista** y **enfoque**
- Navega por tus proyectos desde la barra lateral

### 📊 Vistas adaptadas a tu flujo
| Vista | Descripción |
|-------|-------------|
| **Mi día** | Las tareas de hoy. Foco total en lo que importa ahora |
| **Todas las tareas** | Vista completa con filtros por estado, prioridad y categoría |
| **Próximamente** | Tareas con fecha límite en los próximos días |
| **Kanban** | Tablero de columnas por estado dentro de cada proyecto |
| **Analíticas** | Gráficos de productividad semanal y distribución de tareas |

### 🎨 Estados personalizados
Define tus propios estados de flujo de trabajo (por ejemplo: *En revisión*, *Bloqueado*, *QA*, *Listo para deploy*). Se organizan en tres fases — **No iniciado**, **En proceso** y **Terminado** — que determinan cómo se calculan tus estadísticas.

### ⌨️ Diseñado para el teclado
TaskMaster se puede usar casi completamente sin ratón:

| Atajo | Acción |
|-------|--------|
| `N` | Nueva tarea |
| `Ctrl + 1 … 5` | Navegar entre vistas |
| `Ctrl + ,` | Abrir ajustes |
| `Ctrl + B` | Mostrar / ocultar barra lateral |
| `Ctrl + K` | Buscar tareas |
| `/` | Enfocar el buscador |
| `?` | Ver todos los atajos |
| `Esc` | Cerrar modal o menú |
| Clic derecho | Menú contextual de tarea |

### 👥 Equipos y colaboración *(requiere cuenta)*
- Convierte cualquier proyecto en un **espacio de equipo**
- Invita colaboradores con roles: **Administrador**, **Editor** o **Colaborador**
- Las tareas del equipo se sincronizan en tiempo real entre todos los miembros
- Registro de actividad del equipo

### 🔒 Privacidad primero
- Todo se guarda **localmente** en tu dispositivo (SQLite). Cero acceso externo sin tu permiso
- La cuenta es completamente **opcional** — la app funciona al 100% sin conexión
- Sin publicidad, sin rastreadores, sin venta de datos
- Exporta todos tus datos en JSON cuando quieras

### 🔄 Actualizaciones automáticas
TaskMaster se actualiza solo cuando hay una nueva versión disponible. Recibes una notificación dentro de la app y decides cuándo instalar.

## Descarga e instalación

### Requisitos
- Windows 10 / 11 (64 bits)

### Pasos
1. Ve a la sección [**Releases**](../../releases/latest)
2. Descarga el archivo `TaskMaster_x.x.x_x64-setup.exe`
3. Ejecuta el instalador y sigue los pasos
4. TaskMaster se instala y queda listo para usar

> **¿Aviso de Windows Defender?** Es normal en apps que no están firmadas con un certificado de pago. Haz clic en *"Más información"* → *"Ejecutar de todas formas"* para continuar.
